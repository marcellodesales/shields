Here are a few other parameters you can use:

?label=healthinessesOverride the default left-hand-side text
?logo=data:image/png;base64,… Insert logo image (≥ 14px high)
?logoWidth=40 Set the horizontal space to give to the logo
?link=http://abc.xyz&link=http://shields.io Specify what clicking on the left/right of a badge should do (esp. for social badge style)

Image Examples

For the image, do the following:
  • Download an image (transparent PNG preferred)
  • Edit grayscale: http://osxdaily.com/2013/09/30/convert-an-image-into-black-white-with-preview-in-mac-os-x/
  • Generate base64: https://www.base64-image.de/
  • URL Encode it: http://meyerweb.com/eric/tools/dencoder/
  • Place the encoded value in the logo=PLACE_HERE


    		•
    For the badge color, do the following:
  • Get the Hex value of a given pixel color of an image: http://imagecolorpicker.com/en
  • Look for a variant of the HEX value http://www.hex-colors.net/f2f2f2-hex-color/
  • Place the value in the 6 values before "svg"
    		•
